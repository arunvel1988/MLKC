About NGINX Ingress Controller

The NGINX Ingress Controller is a Kubernetes-native way to manage external access to your services, offering features like load balancing, SSL termination, and name-based virtual hosting.

It uses NGINX as a reverse proxy and load balancer to route HTTP(S) traffic into your Kubernetes cluster efficiently.

Learn more and get started on the official NGINX Ingress GitHub page.
